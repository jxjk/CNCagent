# CNC Agent 用户界面使用说明

## 简介
CNC Agent 现在提供了一个用户友好的Web界面，方便用户上传PDF图纸并生成FANUC NC程序。

## 访问方式
- 打开浏览器访问: `http://localhost:5000/`
- 界面将展示一个简洁的表单用于上传PDF和输入加工描述

## 功能特性
- **PDF上传**: 支持上传PDF格式的工程图纸
- **加工描述**: 用户可以输入加工需求（如钻孔、铣削等）
- **比例尺设置**: 可选的比例尺参数，用于调整图纸尺寸
- **实时预览**: 生成的NC程序将直接显示在页面上
- **文件下载**: 一键下载生成的NC程序文件

## 使用步骤
1. 在"PDF图纸文件"字段中选择您的PDF图纸文件
2. 在"加工描述"文本框中输入加工要求
   - 例如："请加工一个直径10mm的孔，深度5mm"
   - 或："铣削一个长20mm宽15mm的矩形槽，深度3mm"
3. 如需要，调整"图纸比例尺"参数（默认为1.0）
4. 点击"生成NC程序"按钮
5. 生成的NC代码将显示在页面下方
6. 如需保存，可点击"下载NC文件"按钮

## 支持的加工类型
- 钻孔 (drilling): 识别为"钻孔"、"hole"、"孔"等关键词
- 铣削 (milling): 识别为"铣削"、"milling"、"铣"等关键词
- 车削 (turning): 识别为"车削"、"turning"、"车"等关键词

## 技术细节
- 前端: HTML, CSS, JavaScript
- 后端: Flask框架
- 界面采用响应式设计，支持桌面和移动设备
- 使用AJAX实现异步文件上传和处理

## API端点
- 主页: `GET /`
- 健康检查: `GET /health`
- NC生成: `POST /generate_nc`
- 文件下载: `GET /download_nc/<file_path>`
- API接口: `POST /api/generate`

## 注意事项
- 确保上传的PDF文件包含清晰的几何图形
- 加工描述越详细，生成的NC程序越精确
- 如遇到Tesseract OCR相关错误，请确保已正确安装Tesseract引擎
